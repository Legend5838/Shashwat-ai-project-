# AI Smart Reply Automation System

A B.Tech mini/major project using Python Flask and SQLite.

## Features
- User registration and login
- AI-style smart reply generation
- Tone selection: Formal, Friendly, Short
- Reply history saved in SQLite database
- Clean dashboard UI

## Tech Stack
- Python 3
- Flask
- SQLite
- HTML/CSS

## How to Run

1. Install Python 3.
2. Open terminal in this folder.
3. Run:
```bash
pip install -r requirements.txt
python app.py
```
4. Open browser:
```text
http://127.0.0.1:5000
```

## Default Notes
This project uses a simple rule-based AI-style reply generator, so it runs without paid API keys.
You can later connect OpenAI/Gemini API if required by college.
